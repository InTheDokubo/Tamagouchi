# たまごち！～ゆるふわ育成ライフ～

育成・デッキ構築・ローグライト進行を組み合わせたブラウザカードゲームです。

## ローカル起動

```sh
npm start
```

`http://localhost:3000` を開いてください。

## 基本ルール

- 1ターンの基本行動回数は1回、基本ドローは4枚、手札上限は9枚です。
- 「続けて行動」は使用時に行動権を1回復します。
- 「1回のみ」のカードは、使用するとその戦闘中は山札へ戻りません。
- カード自身の反動ダメージではHP1未満にならず、反動だけで戦闘不能にはなりません。
- 回復カードは最大HPに対する割合で回復し、小数点以下は切り上げます。カード強化で回復割合も上昇します。
- 物理攻撃を同じターンに重ねるとコンボ補正が上がります。多段攻撃は各ヒットごとにコンボが進み、後半ほど威力が上がります。
- 魔力型カードは固有資源「一時魔力」を獲得・消費します。一時魔力は敵を倒しても持ち越し、ショップまたは秘伝の改造へ到達すると0になります。
- 炎上は敵行動後にダメージを与えて1減少、脆弱は次に受ける一撃を1.5倍、弱化は敵の次の攻撃を25%下げます。

## ジョブ

各ジョブは5枚の専用初期デッキを持ちます。敗北後の継承強化は初期能力へ別途加算されます。

### 体力型 🐻

- 初期能力: 最大HP65・攻撃5・魔力5
- 初期デッキ: パンチ、防御2枚、絆創膏、ボディプレス
- 生命の殻: 戦闘開始時、最大HPの10%（最低6）のブロックを得ます。
- 自然治癒: 各ターン終了時、最大HPの6%（最低4・最大10）を回復します。

#### 体力型の設計

体力型は「高い最大HP・回復・ブロックで堅実に戦う状態」と、「蓄えたHPを大ダメージへ変えるリスク状態」を往復するジョブです。

- 堅実側: 回復でHPを蓄え、生命の殻や防御カードで敵の大技を受けます。最大HPが高いほど自然治癒とHP参照カードも強くなります。
- リスク側: ボディプレス、生命転換、プレスはHPを消費して大きなダメージを出します。HP消費はHP1で止まりますが、その直後の敵攻撃には注意が必要です。
- 復帰: 絆創膏、底力、大回復などの割合回復で安全圏へ戻ります。最大HPを伸ばしても回復カードの価値が落ちず、再びHPを火力へ変換できます。
- 判断軸: 敵の予告攻撃、手札の回復量、現在HPから「守る・回復する・使い切る」を毎ターン選びます。

### 攻撃型 🐯

- 初期能力: 最大HP50・攻撃8・魔力5
- 初期デッキ: パンチ、ジャブ、防御、キック、激怒
- 先手必勝: 戦闘の最初のターンだけ行動回数が2回になります。
- 連撃の呼吸: 各ターン、物理攻撃のコンボが初めて3以上になったとき、行動権を1回復してカードを1枚引きます。1ターンに1回だけ発動します。
- クロスカウンター: 敵の攻撃を受けるとき、10%の確率で完全回避し、その攻撃の火力1.5倍で反撃します。

#### 攻撃型の設計

攻撃型は「起点・連携・決め技」の順にカードをつなぎ、1ターンへ火力を集中させるテンポ型ジョブです。

- 起点: ジャブやフェイントは低威力ですが行動権を消費せず、コンボと脆弱を準備します。
- 連携: 連続拳や乱れ打ちは1枚で複数ヒットを稼ぎ、連撃の呼吸を早く発動させます。
- 決め技: キック、本気パンチ、とどめの一撃などを高いコンボ補正や脆弱へ重ねます。
- リスク: 軽い攻撃だけでは総火力が不足し、大技を早く切るとコンボ補正を失います。敵の予告を見ながら攻め切るか、防御を挟むか判断します。
- デッキ構築: 起点カードを増やしすぎず、コンボを稼ぐカード・手札を補充するカード・決め技の比率を整えることが重要です。

### 魔力型 🦉

- 初期能力: 最大HP50・攻撃5・魔力8
- 初期デッキ: パンチ2枚、防御、プチ魔法、バリア
- 一時魔力: 魔力型カードごとに1～4獲得します。敵を倒しても次の戦闘へ持ち越され、ショップまたは秘伝の改造へ到達すると0になります。
- 魔力型の設計: 通常魔力を参照する攻撃・防御は扱いやすく、一時魔力を貯めて魔道書による手札循環、魔力解放、時間操作、残響など固有の特殊効果へ変換します。貯め続けるか、目の前の戦闘で使うかが判断軸です。
- 秘奥術: 戦闘画面の一時魔力パネルから、行動権を使わず発動できます。各術式は1戦闘1回です。
  - 魔導転写（一時魔力3）: カードを2枚引きます。
  - 位相転換（一時魔力5）: 次の敵攻撃を完全無効化し、その火力の半分（最大8）を一時魔力へ変換します。
  - 時間圧縮（一時魔力8）: 行動権を1回復し、カードを2枚引きます。

「共通」はどのジョブでも獲得候補になります。各型のカードは、そのジョブの報酬やショップを中心に登場します。表は未強化時の効果です。

## 秘伝の改造モード

10体撃破ごとに出現し、100BPでカード1枚へ固有能力を追加します。同じカード名には同じ秘伝能力が割り当てられます。

| 秘伝能力 | 追加効果 | 対象カード |
|---|---|---|
| 破砕の型 | 使用時、敵へ脆弱1を付与。 | パンチ、キック、タックル、羅刹撃、燕返し、とどめの一撃、崩し打ち、ジャブ、フェイント、生命転換、ボディプレス、ドラゴン拳、吸血撃、剣聖の見切り |
| 連撃奥義 | ヒット数ぶん、さらにコンボを加算。 | 連続拳、乱れ打ち、無限連斬 |
| 輪廻刻印 | 「1回のみ」を失い、使用後は捨て札へ戻る。 | 本気パンチ、プレス、城門 |
| 不動結界 | そのカードで得たブロックを次のターンへ持ち越す。 | 防御、構える、不屈、要塞化、鋼の意志、報復要塞 |
| 反攻装甲 | 使用時、この戦闘中の反撃+2。 | トゲの鎧、神盾イージス、シールドバッシュ |
| 生命変換 | 最大HPを超えた回復量をブロックへ変換。 | 絆創膏、深呼吸、底力 |
| 再生循環 | 使用時、さらに1枚引く。 | 大回復、活力 |
| 静心の守り | 使用時、ブロック6を得る。 | 思考加速、声援、集中、激怒、ビルドアップ、鬼神化、巨人の体、狂戦士、不死身、限界突破、世界樹の加護 |
| 先読み術式 | 使用時、さらに1枚引く。 | 瞑想 |
| 無拍子 | 使用時、行動権を1回復。 | 仕込み、再構築 |
| 生贄回路 | 一時魔力の獲得量が2倍になる代わり、使用後はその戦闘から除外。 | プチ魔法、雷撃、バリア、魔力装甲 |
| 虚無蒸留 | 攻撃前に敵の全ブロックを消滅させ、量に応じて一時魔力へ変換（最大6）。 | 炎魔法、オーバーロード、メテオ、ブラックホール |
| 異常式 | 使用時、弱化・炎上4・脆弱1のどれかをランダムで追加。 | 魔力充填、氷結魔法、灼熱、絶対零度 |
| 逆行支払い | 消費した一時魔力の半分を、次の自分ターン開始時に取り戻す。 | 魔道書、魔力解放、時渡り、残響魔法、因果反転 |
| 未来複製 | 各戦闘の初回使用時、秘伝なし・一回限りの複製を捨て札へ生成。 | 未来予知 |
