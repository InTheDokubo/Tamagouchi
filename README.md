# たまごち！～ゆるふわ育成ライフ～

育成・デッキ構築・ローグライト進行を組み合わせたブラウザカードゲームです。

## ローカル起動

```sh
npm start
```

`http://localhost:3000` を開いてください。

## 基本ルール

- 1ターンの基本行動回数は1回、基本ドローは4枚、手札上限は9枚です。
- 「続けて行動」は使用時に行動権を1回復します。
- 「1回のみ」のカードは、使用するとその戦闘中は山札へ戻りません。
- カード自身の反動ダメージではHP1未満にならず、反動だけで戦闘不能にはなりません。
- 物理攻撃を同じターンに重ねるとコンボ補正が上がります。多段攻撃は各ヒットごとにコンボが進み、後半ほど威力が上がります。
- 魔法を同じターンに連続使用すると、2枚目以降に詠唱蓄積ボーナスが加わります。
- 炎上は敵行動後にダメージを与えて1減少、脆弱は次に受ける一撃を1.5倍、弱化は敵の次の攻撃を25%下げます。

## ジョブ

全ジョブ共通の初期能力は最大HP50・攻撃5・魔力5で、初期デッキは「パンチ」2枚と「防御」1枚です。ここへジョブ固有の能力と2枚が追加されます。敗北後の継承強化は別途加算されます。

### 体力型 🐻

- 初期能力: 最大HP60・攻撃5・魔力5
- 初期追加カード: 絆創膏、タックル
- 生命の殻: 戦闘開始時、最大HPの10%（最低6）のブロックを得ます。
- 自然治癒: 各ターン終了時、最大HPの6%（最低4・最大10）を回復します。

### 攻撃型 🐯

- 初期能力: 最大HP50・攻撃8・魔力5
- 初期追加カード: キック、激怒
- 先手必勝: 戦闘の最初のターンだけ行動回数が2回になります。

### 魔力型 🦉

- 初期能力: 最大HP50・攻撃5・魔力8
- 初期追加カード: プチ魔法、バリア
- 詠唱蓄積: 毎ターン開始時、次の魔法ダメージへ+2を蓄積します。魔法を使うと蓄積分をすべて消費します。
- 魔力循環: 「1回のみ」ではない魔法カード、または魔力型カードを使ったとき、30%の確率で捨て札へ送らず手札へ戻します。手札が9枚なら捨て札へ送ります。「1回のみ」のカードには発動しません。

「共通」はどのジョブでも獲得候補になります。各型のカードは、そのジョブの報酬やショップを中心に登場します。表は未強化時の効果です。

## 秘伝の改造モード

10体撃破ごとに出現し、100BPでカード1枚へ固有能力を追加します。同じカード名には同じ秘伝能力が割り当てられます。

| 秘伝能力 | 追加効果 | 対象カード |
|---|---|---|
| 破砕の型 | 使用時、敵へ脆弱1を付与。 | パンチ、キック、タックル、羅刹撃、燕返し、とどめの一撃、崩し打ち、ジャブ、フェイント、生命転換、ボディプレス、ドラゴン拳、吸血撃、剣聖の見切り |
| 連撃奥義 | ヒット数ぶん、さらにコンボを加算。 | 連続拳、乱れ打ち、無限連斬 |
| 輪廻刻印 | 「1回のみ」を失い、使用後は捨て札へ戻る。 | 本気パンチ、炎魔法、メテオ、プレス、城門 |
| 魔力残響 | 使用後、次の魔法ダメージへ+4。 | プチ魔法、雷撃、オーバーロード、魔力解放、絶対零度 |
| 追炎術式 | 使用時、炎上3を追加。 | 氷結魔法、灼熱、ブラックホール |
| 不動結界 | そのカードで得たブロックを次のターンへ持ち越す。 | 防御、バリア、構える、不屈、要塞化、鋼の意志、絶対防御、因果反転、報復要塞 |
| 反攻装甲 | 使用時、この戦闘中の反撃+2。 | 魔力装甲、トゲの鎧、神盾イージス、シールドバッシュ |
| 生命変換 | 最大HPを超えた回復量をブロックへ変換。 | 絆創膏、深呼吸、底力 |
| 再生循環 | 使用時、さらに1枚引く。 | 大回復、活力 |
| 静心の守り | 使用時、ブロック6を得る。魔力循環の発動回数は増やさない。 | 思考加速、声援、集中、激怒、魔力充填、ビルドアップ、鬼神化、巨人の体、狂戦士、残響魔法、不死身、限界突破、世界樹の加護 |
| 先読み術式 | 使用時、さらに1枚引く。 | 瞑想、時渡り |
| 無拍子 | 使用時、行動権を1回復。 | 仕込み、再構築、魔道書、未来予知 |
